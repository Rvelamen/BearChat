BearChat是一个支持Web端调用MCP服务、多个LLM提供商的Web客户端。

# 🌟 Key Features

- 🤖 多种大语言模型供应商支持
- ⚙️ 支持MCP(Model Context Protocol) Server
- 💬 支持MCP Client
- 📝 支持工具调用
- 📊 支持外部接口一键转MCP tool
- 🎯 提供Chrome 插件

# ⚙️ Start
```
pip install -r requirment.txt
python main.py
```
then
```
visit http://127.0.0.1:8888
```
